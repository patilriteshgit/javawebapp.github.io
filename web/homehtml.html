JAVA WEB APPLICATION FORM USING JDBC

This is the Home page of this project which directs to the registration page which takes input from the user and by form action submit & method post it sends the data to the HTTPservlet in which by the do post values are fetched and displayed and then processed to store in MySQL DB by JDBC Connection. If the same name & password entered again request will directed to login successfull JSP page. In login jsp form field ,the username which needs to be deleted can be provided and the request will be directed to deletusername.jsp in which by the jstl sql tags upadte operation is perrformed.
